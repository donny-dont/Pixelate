Pixelate

A UI library for creating amazing web applications.

Currently {{version}}

Designed for modern web applications.

Pixelate allows you to create web applications that rival native applications. It provides a number of widgets that web applications need, allowing you to pick and choose what's important for your application. Elements are themed to give a flat look and can be customized according to your own look and feel.

All of this leaves you to focus on making your application great.

Built with Polymer

Leveraging web components for next generation web applications.

Using Dart

The modern web language for building scalable web applications.

Pixelate is open source. It's hosted, developed, and maintained on GitHub.
