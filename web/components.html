<px-grid-panel>

Defines a grid for laying out elements.

A [GridPanel] is defined through the use of [RowDefinitions], and [ColumnDefinitions] which specify the dimensions of the grid. The placement of individual elements is done by setting data attributes specifying the column and row that contains the element. This allows the positioning to be specified in markup rather than CSS.
